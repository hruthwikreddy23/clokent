# Clokent

Clokent is a full-stack, multi-location shift and attendance management system designed to centralize clock-in/clock-out workflows across distributed library locations.

## Tech Stack
- Frontend: React (Vite), Tailwind CSS
- Backend: Python (Django / DRF) — planned
- Database: PostgreSQL — planned

## Getting Started (Frontend)
```bash
cd frontend
npm install
npm run dev
